# 📅 Wall Calendar App

A modern and interactive wall calendar web application built using React. Users can view dates, navigate between months, and add custom events with color coding.

---

## 🚀 Live Demo

🔗 https://your-site-link.netlify.app

---

## ✨ Features

* 📆 Monthly calendar view
* ⬅️➡️ Navigate between months
* 📍 Highlight current day
* 📝 Add custom events
* 🎨 Color-coded events
* ⚡ Fast and responsive UI

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/your-username/wall-calendar.git
```

Go to project folder:

```
cd wall-calendar
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```

---

## 📦 Build for Production

```
npm run build
```
